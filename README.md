# niftyuwu.github.io

Personal GitHub Pages repository for niftyuwu projects.

## Cydia repo

Add this source in Cydia:

```text
https://niftyuwu.github.io/repo/cydia/
```

Current packages:

- **DDGSearch** — redirects iOS 6 Safari Google searches to DuckDuckGo HTML.
- **Triple Silent** — Activator action for toggling Silent Mode on iOS 6.
- **slidetouwu** — small lock screen text tweak.
