Triple Silent

Toggle Silent Mode from Activator.

What it does

Triple Silent adds an Activator action that toggles Silent Mode. It is useful for older iPhones with a broken mute switch.

  • No background daemon
  • No settings app clutter
  • Built for iOS 6-era devices
  • Works through Activator, so the Home button gesture does not quit the current app

Setup

After installing, open:

Settings → Activator → Anywhere → Home Button → Triple Press

Choose Triple Silent.

Dependencies

  • MobileSubstrate
  • Activator
