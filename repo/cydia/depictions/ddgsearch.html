DDGSearch

Redirect Safari searches to DuckDuckGo HTML.

What it does

DDGSearch keeps the normal iOS 6 Safari search engine menu untouched, but redirects Google search URLs to the lightweight DuckDuckGo HTML interface.

  • Designed for iOS 6 Safari
  • Uses DuckDuckGo HTML for better old-browser compatibility
  • No system plist patching
  • Works while Google is selected in Safari settings

Setup

Install the tweak, keep Settings → Safari → Search Engine set to Google, then search from Safari’s address bar.

Dependencies

  • MobileSubstrate
